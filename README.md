GO-Promo

Веб-приложение для проведения промо-акций с регистрацией чеков и автоматическим розыгрышем победителей.

Возможности

- Регистрация участников по номеру чека
- Проверка корректности номера чека
- Нормализация телефонных номеров
- Защита от повторной регистрации одного и того же чека
- Хранение данных в Google Sheets
- Панель администратора
- Авторизация администратора
- Автоматический выбор победителя
- Просмотр списка участников
- Просмотр списка победителей
- Удаление победителей с возвратом участника в общий список

Используемые технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Apps Script
- Google Sheets

Настройка Google Sheets

1. Создайте новую Google Таблицу.
2. Откройте:
   - Расширения → Apps Script.
3. Вставьте содержимое файла "google_apps_script.js".
4. Сохраните проект.
5. Выполните функцию "setup()".
6. Создайте веб-приложение:
   - Развертывание → Новое развертывание.
   - Тип: Веб-приложение.
   - Доступ: Все.

После публикации скопируйте URL веб-приложения.

Настройка проекта

Откройте файл "config.json":

{
  "googleScriptUrl": "ВАШ_URL_APPS_SCRIPT",
  "googleSheetId": "",
  "startDate": "2026-05-22T00:00:00",
  "endDate": "2026-06-10T23:50:59",
  "drawDate": "2026-07-02T00:00:00"
}

Укажите URL опубликованного Apps Script.

Настройка администратора

В редакторе Apps Script выполните:

PropertiesService.getScriptProperties().setProperty(
  "ADMIN_LOGIN",
  "your_login"
);

PropertiesService.getScriptProperties().setProperty(
  "ADMIN_PASSWORD",
  "your_password"
);

После этого администратор сможет войти в панель управления.

Безопасность

- Проверка обязательных полей

- Проверка формата номера чека
- Защита от дублирования чеков
- Авторизация администратора
- Защита от одновременных операций через LockService
