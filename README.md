# GO Promo

Система проведения промо-акций и розыгрышей призов с регистрацией чеков, административной панелью и хранением данных в Google Sheets.

## Возможности

### Для участников

- Регистрация чеков через веб-форму
- Проверка корректности номера чека
- Автоматическая нормализация номера телефона
- Защита от повторной регистрации одного и того же чека
- Отображение таймера до даты розыгрыша
- Публикация списка победителей

### Для администратора

- Авторизация через Google Apps Script
- Просмотр всех участников
- Поиск по чеку, имени и телефону
- Проведение случайного розыгрыша
- Управление победителями
- Сброс статуса победителя
- Настройка периода регистрации
- Временное закрытие регистрации
- Просмотр статистики

## Проверка данных

### Чек

Поддерживаются только номера формата:

```text
000081XXXXXX
```

где:

- первые 6 цифр обязательно `000081`
- всего 12 цифр

### Телефон

Автоматически удаляются:

```text
+373
373
```

После нормализации номер должен содержать ровно 8 цифр.

Примеры:

```text
+37377712345 → 77712345
37377712345  → 77712345
77712345     → 77712345
```

## Настройка Google Apps Script

1. Создать Google Таблицу.
2. Открыть:

```text
Расширения → Apps Script
```

3. Вставить код из:

```text
google_apps_script.js
```

4. Выполнить функцию:

```javascript
setup();
```

5. Развернуть как Web App:

```text
Новое развертывание
→ Веб-приложение
→ Доступ: Все
```

6. Получить URL веб-приложения.

7. Указать URL в:

```json
{
  "googleScriptUrl": "ВАШ_URL"
}
```

## Настройка администратора

В Script Properties необходимо задать:

```text
ADMIN_LOGIN
ADMIN_PASSWORD
```

Пример:

```text
ADMIN_LOGIN=admin
ADMIN_PASSWORD=StrongPassword123
```

## Функции администратора

### Розыгрыш победителя

Система случайно выбирает участника, который ещё не выигрывал.

После выбора:

- участник получает статус "Победитель"
- запись переносится в лист "Победители"
- действие записывается в журнал

### Сброс победителя

Позволяет:

- удалить запись из списка победителей
- вернуть участника в общий список
- сохранить действие в журнале
