Инструкция по настройке Google Sheets 🗂️

API не настроено в файле конфигурации. Сейчас приложение использует демо-данные ( Mock ).

Чтобы сайт начал сохранять данные в Google Таблицы, необходимо:

  1. Создать Google Таблицу и назвать новую вкладку Участники и еще одну Победители.
  2. В верхнем меню выбрать: Расширения -> Apps Script.
  3. Скопировать код из файла google_apps_script.js (он сгенерирован для вас) и вставить в редактор.
  4. Нажать кнопку Начать развертывание -> Новое развертывание.
  5. Выбрать тип: Веб-приложение. В доступе указать "Все".
  6. Скопировать полученный URL и вставить его в файл config.json (В поле googleScriptUrl).
